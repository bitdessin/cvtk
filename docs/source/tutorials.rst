User Guides
###########


This tutorial explains how to use **cvtk** to perform basic computer vision tasks.
The procedures presented in this tutorial will primarily use the commands provided by the **cvtk** package.
However, **cvtk** is a Python package and can be incorporated into your own Python scripts for use as an API.
Please refer to the reference for the functions introduced in each tutorial for details.
