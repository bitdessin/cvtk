Tutorials
#########


This tutorial explains how to use **cvtk** to perform basic computer vision tasks, 
including object classification, object detection, instance segmentation, 
and generating web applications for predictions or integration with Label Studio for machine learning-assisted annotation.
This tutorial primarily demonstrates the commands provided by the **cvtk** package. 
However, by editing the scripts generated by **cvtk**,
you can customize their behavior to suit your specific needs.
