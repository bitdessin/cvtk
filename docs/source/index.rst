The Documentation for cvtk
##########################

The **cvtk** (Computer Vision Toolkit) package offers a set of command-line interfaces
for common computer vision tasks,
including image processing, object classification, and image classification.
Built on top of popular libraries such as **PyTorch** and **MMDetection**,
**cvtk** provides a simple and intuitive interface for performing complex tasks.
This makes it accessible for users
who are beginners in both computer vision and the Python programming language,
enabling them to quickly start their projects in computer vision.
